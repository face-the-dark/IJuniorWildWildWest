# Дикий-дикий запад

Игрок управляет бойцом от третьего лица, сражающимся на замкнутой арене против волн врагов. 

Основное его оружие - огнестрельное. 

Враги спавнятся за краями арены. 

После гибели игрока сцена перезапускается и сбрасывается счетчик волн. 

При победе появляется специальное окно - "Victory".

## Детальное описание

**Игрок**

- Управление:
  - Передвижение с помощью `W/A/S/D`.
  - Обзор камеры с помощью мыши.
- Оружие:
  - Стартовое оружие - огнестрельное.
  - Стрельба по `ЛКМ`.
- Здоровье:
  - При получении урона уменьшается `HP`.
  - При _0_ `HP` игрок погибает.

**Враги**

  - Враги появляются за границей арены.
  - Идут к игроку и атакуют его при приближении.

**Условие прохождение волны**

  - Убить всех врагов.
  - Новая волна не начинается пока не закончилась предыдущая.

**Условие победы**

  - Пройти _3_ волны врагов и победить боссов на каждой из волн.

**Задачи со звездочкой**

  - Сделай густой дым в котором ничего не видно. Враги появляются из этого дыма, игрок не должен видеть их появление. Визуально создается ощущение, будто враги выходят из плотного облака.
  - Второе оружие - ближнего боя. Переключение на него (например, клавиша `Q`). Переключение на основное оружие - на ту же кнопку.
  - Сделать босса - в конце волны появляется враг с большим количеством здоровья.
